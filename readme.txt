== Description ==

Simple Post Meta is awsome plugin for show post id, post aurthor, post type, post slug, and all meta information related to post, page.

<br />

<strong>Plugin Features</strong><br />

* Show Meta ID.
* Show Aurthor Name.
* Show Post Type.
* Show Post Slug.
* Show All Meta Information for specific page and posts. 


== Installation ==

1. Install as regular WordPress plugin.<br />
2. Go your plugin setting via WordPress Dashboard and find "<strong>Simple Post Meta</strong>" activate it.<br />

After activate plugin you will see the meta box in page and post and post_types.<br />

<br />
<strong>How to display Simple Post Meta ?</strong><br />

See the Meta Box in all post and page.
